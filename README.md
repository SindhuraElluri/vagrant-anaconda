# vagrant-anaconda

```
$ git clone https://github.com/tomohiro1221/vagrant-anaconda.git
$ cd vagrant-anaconda
$ vagrant up # this might take more than a few minutes

$ vagrant ssh
# inside VM
$ python
>>> import numpy # it should work without errors
$ ipython notebook --ip=0.0.0.0 # ipython notebook @ 127.0.0.1:8888
```
